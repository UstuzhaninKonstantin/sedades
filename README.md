# sedades

Typescript [Evades.io](https://evades.io) offline clone

## Installation

1. Clone the repo:

```bash
git clone https://github.com/UstuzhaninKonstantin/sedades.git
```

2. Install dependencies ([yarn](https://yarnpkg.com/)):

```bash
yarn
```

3. Run:

- dev:

```bash
yarn dev
```

- prod:

```bash
yarn prod
```

- build:

```bash
yarn build
```
